# Bilhetes Procarro

App de gestão dos bilhetes do Grupo Procarro para os jogos do FC Porto no Estádio do Dragão.

- Frontend: React + Vite (PWA)
- Backend: Supabase (base de dados, autenticação, armazenamento dos PDFs)
- Publicação: GitHub Actions → GitHub Pages (automática em cada push ao `main`)

O script da base de dados está em `setup.sql`.
